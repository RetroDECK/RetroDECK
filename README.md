# RetroDECK


RetroDECK is an all-in-one retro gaming platform for Linux-based systems, including handheld gaming PCs (such as the Steam Deck), desktop Linux computers, HTPC setups, and Linux-powered console devices.

Delivered as a self-contained Flatpak application, RetroDECK provides a ready-to-use retro gaming environment right out of the box, no manual setup required. 

By integrating a wide range of components such as emulators, game engines, ports, and built-in tools, RetroDECK brings all your favorite games together in one streamlined, easy-to-use package.
